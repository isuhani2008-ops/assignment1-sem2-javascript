Smart Event Dashboard
Project Description

The Smart Event Dashboard is a simple web application that allows users to create, view, and manage events.
This project demonstrates DOM manipulation, event handling, and dynamic UI creation using JavaScript.

The dashboard includes a form to add events and displays them as cards.

Features

Add new events using a form

Display events as cards

Delete individual events

Clear all events

Add sample events

Keyboard event detection (DOM manipulation section)

Responsive and modern UI

Technologies Used

HTML

CSS

JavaScript (Vanilla JS)

How to Run the Project

Download or copy the project files.

Make sure these files are in the same folder:

index.html

style.css

script.js

Open index.html in a browser.
